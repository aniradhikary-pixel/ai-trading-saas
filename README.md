# AI Trading Signals SaaS

A full-stack crypto trading signal dashboard built with FastAPI, React, SQLite, and Telegram alerts.

## Features
- Multi-coin signal monitoring
- Persistent trade history with SQLite
- Performance dashboard with win rate and profit tracking
- Telegram alert integration
- Automated backend signal scanning

## Tech Stack
- FastAPI
- React
- SQLite
- APScheduler
- Telegram Bot API

## Status
This project is under active development as a real SaaS product and portfolio project.

## Notes
Sensitive credentials and environment-specific settings are managed through environment variables and are not included in the repository.
